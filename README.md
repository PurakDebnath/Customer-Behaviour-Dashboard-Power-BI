# Customer-Behaviour-Dashboard-Power-BI

-This project presents an interactive Power BI Customer Behaviour Dashboard built using retail customer purchase and subscription data.
The main goal is to analyze customer behavior based on age group, subscription status, product category, season, and discounts.

# Project Overview

-Analyzes customer purchasing patterns and subscription trends
-Provides insights on revenue, age groups, and seasonal discounts
-Helps businesses make data-driven sales and marketing decisions
-Business Objective
-Understand customer purchase behavior
-Analyze revenue by product category
-Compare subscriber vs non-subscriber trends
-Identify seasonal discount impact

# Tools and Technologies Used

-Power BI
-Microsoft Excel / CSV
-Power Query
-DAX
-Key Insights
-Total customers: ~3.9K
-Average rating: 3.75
-Average purchase value: $59.76
-Clothing generates highest revenue
-Most customers are non-subscribers
-Discounts are highest in Spring and Fall

# Value Delivered

-Improves understanding of customer behavior
-Supports targeted marketing strategies
-Demonstrates real-world Power BI and DAX skills
-Useful for portfolio and interviews
